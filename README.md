## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Project setup

```bash
$ npm install
```

## Compile and run the project
### Important  
Before running the project, every person who wants to run it must create .env file inside project with these fields:  
```bash
-DATABASE_URL  
-PORT  
-JWT_SECRET  
```
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## About the project

### Features
```bash
User Registration & Authentication  
Users can create an account and log in to access features.  
```

### Post management:
```bash
Create new posts  
Read all posts or individual posts  
Update existing posts  
Delete posts  
Comments  
Users can leave comments on posts  
Read all comments on a post  
```

### Tech stack
```bash
Language: [Node.js]  
Database: [MongoDB]  
Authentication: JWT (JSON Web Tokens)  
```

## Structure
/src  
├── auth        # Handles registration, login, authentication logic  
├── comment     # Handles comment creation, reading comments  
├── posts       # Handles post creation, updating, deleting, reading  
├── shared      # Contains shared utilities, guards, db-tools, or constants  
├── user        # Handles user data, profile management
